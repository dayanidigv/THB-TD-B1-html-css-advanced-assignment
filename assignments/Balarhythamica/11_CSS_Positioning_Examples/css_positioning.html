Relative positioning moves an element relative to its normal position. It still occupies its original space in the flow.

Fixed Positioning

Fixed elements remain in the same position relative to the viewport, even when the page is scrolled.

Try scrolling down the page; the floating button stays fixed.

Sticky Positioning

Sticky elements behave like relative elements until a defined scroll position is reached, then they “stick” in place.

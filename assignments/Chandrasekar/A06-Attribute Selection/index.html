Learn, visualize, and practice all types of CSS attribute selectors

[attribute]

[attribute] Selector

Selects elements that have a specific attribute, regardless of its value.

This box is selected using [data-demo]
<div data-demo="example">This box</div>
[data-demo] {
    background-color: #c8e6c9;
    color: #2e7d32;
}

[attribute=value] Selector

Selects elements with an exact attribute value.

input[type="text"] {
    border: 2px solid #1976d2;
    padding: 5px 10px;
    border-radius: 5px;
}

[attribute~=value] Selector

Selects elements whose attribute value contains a specified word (space-separated).

button[class~="btn"] {
    padding: 10px 15px;
    border-radius: 5px;
    border: none;
    color: white;
    margin: 5px;
}
button.btn-primary { background: #1976d2; }
button.btn-danger { background: #d32f2f; }

[attribute^=value] Selector

Selects elements whose attribute value starts with a specific value.

Visit Secure Site 
a[href^="https://"] {
    color: #ff5722;
    font-weight: bold;
    text-decoration: none;
}

[attribute$=value] Selector

Selects elements whose attribute value ends with a specific value.

img[src$=".png"],
img[src$=".jpg"] {
    width: 20%;
    height: 20%;
    object-fit: cover;
    border-radius: 10px;
}
.image-gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
    gap: 15px;
}
.image-box {
    width: 20%;
    height: 20%;
    overflow: hidden;
    border-radius: 10px;
    box-shadow: 0 3px 8px rgba(0,0,0,0.2);
}

[attribute*=value] Selector

Selects elements whose attribute value contains a specified substring.

This paragraph is highlighted because class contains "highlight".

Another paragraph using [class*="highlight"].

p[class*="highlight"] {
    background: #fff9c4;
    border-left: 4px solid #fbc02d;
    padding: 5px;
}
